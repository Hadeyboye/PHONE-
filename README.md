# AI Phone Tracker

Production-ready Next.js app with **real Twilio Lookup v2** integration, interactive maps, and bulk processing.

## Setup
1. `npm install`
2. Copy `.env.local` and add your Twilio credentials.
3. `npx prisma generate && npx prisma db push`
4. `npm run dev`

**Legal**: Use only with consent. Twilio provides carrier/intelligence data only.

Ready for Vercel deployment.
